==============
DesktopKud1FZX
==============

This package contains the fonts from Proxima Software's "Desktop" expansion pack
"Klub Uzivatelu Desktopu 1", now converted to the "FZX" standard format.

Since "Desktop" is currently released as "public domain", the same distribution
also applies to these converted fonts.
