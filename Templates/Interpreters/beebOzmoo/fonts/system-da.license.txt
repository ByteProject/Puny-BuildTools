system-da.fnt: Ozmoo version, for Danish language, of the original Commodore 64 system font

License: 

This font is a version of the Commodore 64 system font, modified to include
Danish accented characters. It may be freely used with the Z-code
interpreter Ozmoo, including embedding the font in games built with Ozmoo.
We do not own the copyright for this font. The base font is available in
every Commodore 64 anyway, and the modifications (the accented characters)
are provided for free.
